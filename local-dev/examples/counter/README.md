# Counter Contract Example

シンプルなカウンターコントラクトのサンプルです。

## コンパイル

```bash
compact compile counter.compact out/
```

## 出力ファイル

```
out/
├── index.js          # コントラクトランタイム
├── index.d.ts        # TypeScript 型定義
├── witnesses.js      # ウィットネス定義
└── managed/          # ZK キー等
```

詳細は [../../example-counter](../../../example-counter/) を参照してください。
